Visualization of team slack communications for NIH National Center for Advancing Translational Sciences (NCATS) award project

This communication network shows how team members interact with each other on slack for NIH NCATS award BioMedical Translator project. Link colors represent different types of communication. Node colors are slack user profile colors. You can pan, zoom, reset view, or drag a node to influence layout. You can also click a node or link to select it to see communication details in the info box below the visualization, click it again to deselect it or click another node or link to make other selections.
